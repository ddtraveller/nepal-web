CSS Styling Tutorial

CSS स्टाइलिङ ट्युटोरियल

1. Colors and Background / रङ्ग र पृष्ठभूमि

Change text color and background:

Red Text / रातो पाठ
Yellow Background / पहेंलो पृष्ठभूमि

पाठको रङ्ग र पृष्ठभूमि परिवर्तन गर्ने:

color: #ff0000; /* रातो रङ्ग */ background-color: #ffff00; /* पहेंलो रङ्ग */

2. Borders / बर्डर

Different border styles with their CSS code:

CSS कोड सहित विभिन्न बर्डर शैलीहरू:

.solid-border {
    border: 2px solid red;
    margin: 5px;
    padding: 5px;
    border-radius: 4px;
}
Solid Border / ठोस बर्डर
.dashed-border {
    border: 2px dashed blue;
    margin: 5px;
    padding: 5px;
    border-radius: 4px;
}
Dashed Border / धर्को बर्डर
.dotted-border {
    border: 2px dotted green;
    margin: 5px;
    padding: 5px;
    border-radius: 4px;
}
Dotted Border / थोप्ला बर्डर
.fancy-border {
    border: 4px solid transparent;
    border-image: linear-gradient(45deg, purple, orange) 1;
    margin: 5px;
    padding: 5px;
}
Gradient Border / ग्रेडियन्ट बर्डर

3. Common Hex Colors / सामान्य हेक्स रङहरू

Here are some commonly used hex colors:

यहाँ केही प्रयोग गरिने हेक्स रङहरू छन्:

#FF0000 (Red/रातो)
#00FF00 (Green/हरियो)
#0000FF (Blue/नीलो)
#FFFF00 (Yellow/पहेंलो)
#FF00FF (Magenta/म्याजेन्टा)
#00FFFF (Cyan/सायन)
#000000 (Black/कालो)
#FFFFFF (White/सेतो)

4. CSS Code Examples / CSS कोड उदाहरणहरू

Here are the CSS code examples for borders and interactive elements:

यहाँ बर्डर र इन्टर्याक्टिभ तत्वहरूको लागि CSS कोड उदाहरणहरू छन्:

/* Border Styles / बर्डर शैलीहरू */
.solid-border {
    border: 2px solid red;
    border-radius: 5px;
    padding: 10px;
}

.dashed-border {
    border: 2px dashed blue;
    border-radius: 5px;
    padding: 10px;
}

.dotted-border {
    border: 2px dotted green;
    border-radius: 5px;
    padding: 10px;
}

/* Interactive Elements / इन्टर्याक्टिभ तत्वहरू */
.interactive-element {
    transition: all 0.3s ease;
    cursor: pointer;
}

.interactive-element:hover {
    transform: scale(1.1);
    box-shadow: 0 4px 8px rgba(0,0,0,0.2);
}

/* Gradient Background / ग्रेडियन्ट पृष्ठभूमि */
.gradient-bg {
    background: linear-gradient(45deg, #ff6b6b, #4ecdc4);
    color: white;
    padding: 20px;
}

5. Interactive Elements / इन्टर्याक्टिभ तत्वहरू

Here are different interactive effects with their CSS code:

यहाँ विभिन्न इन्टर्याक्टिभ प्रभावहरू र तिनीहरूको CSS कोड छन्:

.scale-hover {
    width: 100px;
    height: 100px;
    background-color: #3498db;
    transition: transform 0.3s ease;
}

.scale-hover:hover {
    transform: scale(1.1);
}
                
.rotate-hover {
    width: 100px;
    height: 100px;
    background-color: #e74c3c;
    transition: transform 0.3s ease;
}

.rotate-hover:hover {
    transform: rotate(45deg);
}
                
.shadow-hover {
    width: 100px;
    height: 100px;
    background-color: #2ecc71;
    transition: all 0.3s ease;
}

.shadow-hover:hover {
    box-shadow: 0 5px 15px rgba(0,0,0,0.3);
    transform: translateY(-5px);
}

Hover over each box to see its effect in action!
