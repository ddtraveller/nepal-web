jQuery Tutorial

जेक्वेरी ट्युटोरियल

jQuery is a fast, lightweight JavaScript library designed to simplify HTML DOM tree traversal and manipulation.

जेक्वेरी एक छिटो, हल्का जाभास्क्रिप्ट लाइब्रेरी हो जसलाई HTML DOM ट्री ट्राभर्सल र म्यानिपुलेसनलाई सरल बनाउन डिजाइन गरिएको छ।

1. DOM Manipulation - DOM म्यानिपुलेसन

Click the button to change this text

यो पाठ परिवर्तन गर्न बटन क्लिक गर्नुहोस्

$("#changeTextBtn").click(function() { $("#changeMe").text("Text changed with jQuery!"); $("#changeMeNepali").text("जेक्वेरीसँग पाठ परिवर्तन गरियो!"); });

2. Effects - इफेक्टहरू

Fade Effect - फेड इफेक्ट

$("#fadeToggleBtn").click(function() { $("#fadeBox").fadeToggle(); });

Slide Effect - स्लाइड इफेक्ट

$("#slideToggleBtn").click(function() { $("#slideBox").slideToggle(); });

3. Event Handling - इभेन्ट ह्यान्डलिंग

Mouse is over the button!

माउस बटनमाथि छ!

$("#hoverBtn").hover( function() { $(".success-message").show(); }, function() { $(".success-message").hide(); } );

4. Animation - एनिमेसन

$("#animateBtn").click(function() { $("#animateBox").animate({ left: '+=200', width: '+=50', height: '+=50' }, 1000).animate({ left: '-=200', width: '-=50', height: '-=50' }, 1000); });

5. Class Manipulation - क्लास म्यानिपुलेसन

This text can be highlighted

यो पाठ हाइलाइट गर्न सकिन्छ

$("#highlightBtn").click(function() { $("#highlightText, #highlightTextNepali").toggleClass("highlight"); });
