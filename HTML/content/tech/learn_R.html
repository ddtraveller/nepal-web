Introduction to R and RStudio
R र RStudio को परिचय

What is R?
R के हो?

R is a programming language and software environment for statistical computing, data analysis, and graphical visualization.

R एक प्रोग्रामिङ भाषा र सफ्टवेयर वातावरण हो जसलाई तथ्यांक विश्लेषण, सांख्यिकीय गणना, र ग्राफिकल भिजुअलाइजेसनको लागि प्रयोग गरिन्छ।

Installation Guide
इन्स्टलेसन गाइड

Step 1: Install R
पहिलो चरण: R इन्स्टल गर्ने

1. Visit CRAN (Comprehensive R Archive Network)
2. Choose your operating system
3. Download the latest version of R
4. Run the installer and follow the prompts

१. CRAN (Comprehensive R Archive Network) मा जानुहोस्
२. आफ्नो अपरेटिङ सिस्टम छान्नुहोस्
३. R को नवीनतम संस्करण डाउनलोड गर्नुहोस्
४. इन्स्टलर चलाउनुहोस् र निर्देशनहरू पालना गर्नुहोस्

Step 2: Install RStudio
दोस्रो चरण: RStudio इन्स्टल गर्ने

1. Visit RStudio Download Page
2. Download RStudio Desktop (Free Version)
3. Run the installer and follow the prompts

१. RStudio डाउनलोड पेज मा जानुहोस्
२. RStudio Desktop (निःशुल्क संस्करण) डाउनलोड गर्नुहोस्
३. इन्स्टलर चलाउनुहोस् र निर्देशनहरू पालना गर्नुहोस्

RStudio Interface
RStudio इन्टरफेस

RStudio has four main panels:
1. Source Editor (top-left): Write your R code
2. Console (bottom-left): See results and run commands
3. Environment/History (top-right): Track variables and commands
4. Files/Plots/Packages/Help (bottom-right): View outputs and documentation

RStudio मा चार मुख्य प्यानलहरू छन्:
१. सोर्स एडिटर (माथि-बायाँ): R कोड लेख्ने
२. कन्सोल (तल-बायाँ): नतिजाहरू हेर्ने र कमाण्डहरू चलाउने
३. इन्भाइरोनमेन्ट/हिस्टोरी (माथि-दायाँ): भेरिएबलहरू र कमाण्डहरू ट्र्याक गर्ने
४. फाइलहरू/प्लटहरू/प्याकेजहरू/मद्दत (तल-दायाँ): आउटपुट र डकुमेन्टेसन हेर्ने

Creating Your First Chart
तपाईंको पहिलो चार्ट बनाउँदै

Step 1: Create Sample Data
पहिलो चरण: नमूना डाटा सिर्जना गर्ने

# Create sample data months <- c("Jan", "Feb", "Mar", "Apr", "May") sales <- c(150, 200, 175, 225, 250) # Create a data frame data <- data.frame(Month = months, Sales = sales)

Step 2: Create a Bar Chart
दोस्रो चरण: बार चार्ट बनाउने

# Create a basic bar plot barplot(data$Sales, names.arg = data$Month, main = "Monthly Sales", xlab = "Month", ylab = "Sales", col = "skyblue", border = "black")

Step 3: Enhanced Visualization with ggplot2
तेस्रो चरण: ggplot2 सँग उन्नत भिजुअलाइजेसन

# Install and load ggplot2 install.packages("ggplot2") library(ggplot2) # Create an enhanced plot ggplot(data, aes(x = Month, y = Sales)) + geom_bar(stat = "identity", fill = "skyblue") + theme_minimal() + labs(title = "Monthly Sales", x = "Month", y = "Sales") + theme(plot.title = element_text(hjust = 0.5))

Tips for Better Charts
राम्रो चार्टहरूको लागि सुझावहरू

• Use meaningful titles and labels
• Choose appropriate colors
• Consider your audience when selecting chart types
• Add legends when necessary
• Keep designs clean and simple

• अर्थपूर्ण शीर्षक र लेबलहरू प्रयोग गर्नुहोस्
• उपयुक्त रङहरू छान्नुहोस्
• चार्ट प्रकारहरू छान्दा आफ्नो दर्शकहरूलाई ध्यानमा राख्नुहोस्
• आवश्यक हुँदा लेजेन्डहरू थप्नुहोस्
• डिजाइनहरू सफा र सरल राख्नुहोस्

Remember to save your plots using either the Export button in RStudio or the ggsave() function when using ggplot2.

ggplot2 प्रयोग गर्दा आफ्ना प्लटहरू RStudio को Export बटन वा ggsave() फंक्सन प्रयोग गरेर सेभ गर्न नबिर्सनुहोस्।
