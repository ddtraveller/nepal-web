Introduction to Generative AI

Generative AI refers to artificial intelligence systems that can create new content, including text, images, code, and more. These systems are trained on vast amounts of data and can generate human-like outputs based on input prompts.

जेनेरेटिभ एआई को परिचय

जेनेरेटिभ एआई भनेको कृत्रिम बुद्धिमत्ता प्रणालीहरू हुन् जसले नयाँ सामग्री सिर्जना गर्न सक्छन्, जसमा पाठ, चित्रहरू, कोड र अधिक समावेश छन्। यी प्रणालीहरू ठूलो मात्रामा डाटामा प्रशिक्षित छन् र इनपुट प्रम्प्टहरूको आधारमा मानव-जस्तै आउटपुट उत्पन्न गर्न सक्छन्।

Using Claude with Python

Here's how to use Claude through the Anthropic API in Python:

from anthropic import Anthropic
import os

# Initialize the client
anthropic = Anthropic(api_key='your-api-key')

# Create a message
message = anthropic.messages.create(
    model="claude-3-opus-20240229",
    max_tokens=1000,
    messages=[{
        "role": "user",
        "content": "Write a function to calculate the Fibonacci sequence"
    }]
)

# Print the response
print(message.content)

Using Google Vertex AI

Browser-based Interface

Google Vertex AI provides a user-friendly web interface accessible through the Google Cloud Console:

  1. Navigate to console.cloud.google.com and select Vertex AI
  2. Access Language Models under the "Machine Learning" menu
  3. Choose "Test & Use" to access the playground interface
  4. Select your desired model (e.g., PaLM 2, Claude, or other available models)
  5. Configure parameters like:
    • Temperature (0.0 - 1.0)
    • Token limit
    • Top-k
    • Top-p

The playground interface allows you to:

  • Test prompts interactively
  • Save and load prompt templates
  • View token usage and costs
  • Export conversations

ब्राउजर-आधारित इन्टरफेस

Google Vertex AI ले Google Cloud Console मार्फत प्रयोगकर्ता-मैत्री वेब इन्टरफेस प्रदान गर्दछ:

  1. console.cloud.google.com मा जानुहोस् र Vertex AI चयन गर्नुहोस्
  2. "Machine Learning" मेनु अन्तर्गत Language Models मा पहुँच गर्नुहोस्
  3. playground इन्टरफेसमा पहुँच गर्न "Test & Use" चयन गर्नुहोस्
  4. तपाईंको इच्छित मोडेल चयन गर्नुहोस् (जस्तै PaLM 2, Claude, वा अन्य उपलब्ध मोडेलहरू)

Using Claude in Vertex AI with Python

Google Vertex AI provides integration with various LLMs, including Claude. Here's how to use it:

from google.cloud import aiplatform
from vertexai.language_models import TextGenerationModel

# Initialize Vertex AI
aiplatform.init(project='your-project-id')

# Load the model
model = TextGenerationModel.from_pretrained("text-bison@001")

# Generate text
response = model.predict(
    """Write a function to sort an array in Python""",
    temperature=0.2,
    max_output_tokens=1024
)
print(response.text)

Using LangChain with Claude

LangChain is a framework for developing applications powered by language models. Here's an example using Claude with LangChain:

from langchain_anthropic import ChatAnthropic
from langchain.chains import ConversationChain
from langchain.memory import ConversationBufferMemory

# Initialize the Claude chat model
chat = ChatAnthropic(
    anthropic_api_key="your-api-key",
    model_name="claude-3-opus-20240229"
)

# Create a conversation chain with memory
conversation = ConversationChain(
    llm=chat,
    memory=ConversationBufferMemory()
)

# Start a conversation
response = conversation.run("What are the key principles of machine learning?")
print(response)

# Continue the conversation with context
follow_up = conversation.run("Can you provide examples of supervised learning?")
print(follow_up)

LangChain भाषा मोडेलहरूद्वारा संचालित अनुप्रयोगहरू विकास गर्नको लागि एक फ्रेमवर्क हो।
