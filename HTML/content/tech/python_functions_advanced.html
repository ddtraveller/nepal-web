Advanced Python Functions / उन्नत पाइथन फंक्शनहरू

Default Argument Values / पूर्वनिर्धारित तर्क मानहरू

Functions can have default values for arguments:

फंक्शनहरूमा तर्कहरूको लागि पूर्वनिर्धारित मानहरू हुन सक्छन्:

# Basic default arguments / आधारभूत पूर्वनिर्धारित तर्कहरू
def greet(name, greeting="Hello"):
    return f"{greeting}, {name}!"

print(greet("Ram"))           # Hello, Ram!
print(greet("Sita", "Namaste"))  # Namaste, Sita!

# Multiple default values / बहु पूर्वनिर्धारित मानहरू
def create_profile(name, age=20, city="Kathmandu", occupation=None):
    profile = {
        "name": name,
        "age": age,
        "city": city,
        "occupation": occupation
    }
    return profile

Warning: Default values are evaluated once at function definition time.

चेतावनी: पूर्वनिर्धारित मानहरू फंक्शन परिभाषा समयमा एक पटक मूल्यांकन गरिन्छ।

Keyword Arguments / कीवर्ड तर्कहरू

Functions can be called using keyword arguments:

फंक्शनहरूलाई कीवर्ड तर्कहरू प्रयोग गरेर कल गर्न सकिन्छ:

def send_email(to, subject, body, cc=None, bcc=None):
    # Function implementation
    pass

# Using keyword arguments / कीवर्ड तर्कहरू प्रयोग
send_email(
    to="user@example.com",
    subject="Meeting",
    body="Tomorrow at 10 AM",
    cc="manager@example.com"
)

# Mix positional and keyword arguments
send_email("user@example.com", body="Hello!", subject="Hi")

Arbitrary Argument Lists (*args) / स्वेच्छिक तर्क सूची

Functions can accept any number of positional arguments:

फंक्शनहरूले जतिसुकै स्थिति तर्कहरू स्वीकार गर्न सक्छन्:

# Function with *args / *args सहितको फंक्शन
def calculate_sum(*numbers):
    return sum(numbers)

print(calculate_sum(1, 2, 3))       # 6
print(calculate_sum(10, 20, 30, 40))  # 100

# Combining with regular arguments / नियमित तर्कहरूसँग संयोजन
def print_info(name, *hobbies):
    print(f"Name: {name}")
    print("Hobbies:", ", ".join(hobbies))

print_info("Hari", "reading", "hiking", "photography")

Keyword Arguments Dictionary (**kwargs) / कीवर्ड तर्क शब्दकोश

Functions can accept any number of keyword arguments:

फंक्शनहरूले जतिसुकै कीवर्ड तर्कहरू स्वीकार गर्न सक्छन्:

# Function with **kwargs / **kwargs सहितको फंक्शन
def create_user(**user_data):
    return user_data

user = create_user(
    name="Ram",
    age=25,
    city="Pokhara",
    email="ram@example.com"
)

# Combining *args and **kwargs / *args र **kwargs को संयोजन
def process_data(*args, **kwargs):
    print("Args:", args)
    print("Kwargs:", kwargs)

process_data(1, 2, 3, name="Test", value=42)

Unpacking Argument Lists / तर्क सूची खोल्ने

Lists and dictionaries can be unpacked into function arguments:

सूची र शब्दकोशहरूलाई फंक्शन तर्कहरूमा खोल्न सकिन्छ:

# List unpacking / सूची खोल्ने
numbers = [1, 2, 3]
print(calculate_sum(*numbers))  # Same as calculate_sum(1, 2, 3)

# Dictionary unpacking / शब्दकोश खोल्ने
user_data = {
    "name": "Sita",
    "age": 30,
    "city": "Lalitpur"
}
user = create_user(**user_data)

Lambda Expressions / लाम्बडा अभिव्यक्तिहरू

Lambda creates small anonymous functions:

लाम्बडाले साना बेनामी फंक्शनहरू सिर्जना गर्छ:

# Simple lambda / सरल लाम्बडा
square = lambda x: x**2
print(square(5))  # 25

# Lambda with multiple arguments / बहु तर्क सहितको लाम्बडा
multiply = lambda x, y: x * y
print(multiply(3, 4))  # 12

# Lambda in higher-order functions / उच्च-क्रम फंक्शनहरूमा लाम्बडा
numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x**2, numbers))
evens = list(filter(lambda x: x % 2 == 0, numbers))

Documentation Strings / कागजात स्ट्रिङहरू

Docstrings provide function documentation:

डकस्ट्रिङहरूले फंक्शन कागजात प्रदान गर्छन्:

def calculate_interest(principal, rate, time):
    """
    Calculate simple interest.

    Args:
        principal (float): The principal amount
        rate (float): Annual interest rate (as decimal)
        time (float): Time in years

    Returns:
        float: The calculated interest amount

    Example:
        >>> calculate_interest(1000, 0.05, 2)
        100.0
    """
    return principal * rate * time

# Accessing the docstring / डकस्ट्रिङ पहुँच
print(calculate_interest.__doc__)

# Complex function with detailed documentation
def process_transaction(amount, transaction_type="credit", **details):
    """
    Process a financial transaction.

    This function handles various types of financial transactions
    and maintains a detailed log of all operations.

    Args:
        amount (decimal): Transaction amount
        transaction_type (str, optional): Type of transaction
            Defaults to "credit"
        **details: Additional transaction details

    Returns:
        dict: Transaction receipt with all details

    Raises:
        ValueError: If amount is negative
        TypeError: If transaction_type is invalid
    """
    if amount < 0:
        raise ValueError("Amount cannot be negative")
    
    # Function implementation
    return {
        "amount": amount,
        "type": transaction_type,
        "details": details
    }
