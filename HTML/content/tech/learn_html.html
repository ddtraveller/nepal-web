Creating HTML Pages on Windows
Windows मा HTML पेज कसरी बनाउने

Required Tools
आवश्यक टूलहरू

1. Text Editor (Notepad or Visual Studio Code)
2. Web Browser (Chrome, Firefox, or Edge)

१. टेक्स्ट एडिटर (नोटप्याड वा भिजुअल स्टुडियो कोड)
२. वेब ब्राउजर (क्रोम, फायरफक्स, वा एज)

Step 1: Open Notepad
पहिलो चरण: नोटप्याड खोल्नुहोस्

Press Windows + R, type "notepad" and press Enter.

Windows + R थिच्नुहोस्, "notepad" टाइप गर्नुहोस् र Enter थिच्नुहोस्।

Step 2: Write Basic HTML Code
दोस्रो चरण: आधारभूत HTML कोड लेख्नुहोस्

Copy and paste this basic HTML structure:

यो आधारभूत HTML संरचना कपी र पेस्ट गर्नुहोस्:

<!DOCTYPE html> <html> <head> <title>My First Page</title> </head> <body> <h1>Hello World!</h1> <p>This is my first HTML page.</p> </body> </html>

Step 3: Save the File
तेस्रो चरण: फाइल सेभ गर्नुहोस्

1. Click File → Save As
2. Choose a location on your computer
3. Set "Save as type" to "All Files"
4. Name your file with .html extension (example: mypage.html)

१. File → Save As मा क्लिक गर्नुहोस्
२. आफ्नो कम्प्युटरमा स्थान छान्नुहोस्
३. "Save as type" मा "All Files" छान्नुहोस्
४. आफ्नो फाइलको नाम .html एक्सटेन्सनको साथ राख्नुहोस् (उदाहरण: mypage.html)

Step 4: View Your Page
चौथो चरण: आफ्नो पेज हेर्नुहोस्

Double-click the saved HTML file to open it in your default web browser.

सेभ गरिएको HTML फाइलमा डबल क्लिक गर्नुहोस् र यो तपाईंको डिफल्ट वेब ब्राउजरमा खुल्नेछ।

Common Issues and Solutions
सामान्य समस्या र समाधानहरू

Problem: Page shows HTML code instead of rendering
Solution: Make sure file extension is .html, not .txt

समस्या: पेजले HTML कोड देखाउँछ, रेन्डर गर्दैन
समाधान: फाइल एक्सटेन्सन .txt होइन, .html छ भनी सुनिश्चित गर्नुहोस्

Serving Your HTML Page Locally
तपाईंको HTML पेज स्थानीय रूपमा सर्भ गर्दै

Step 1: Open Command Prompt
पहिलो चरण: कमाण्ड प्रम्प्ट खोल्नुहोस्

1. Press Windows + R
2. Type "cmd" and press Enter

१. Windows + R थिच्नुहोस्
२. "cmd" टाइप गर्नुहोस् र Enter थिच्नुहोस्

Step 2: Navigate to Your HTML File Directory
दोस्रो चरण: तपाईंको HTML फाइल डाइरेक्टरीमा जानुहोस्

Use the cd command to navigate to your folder:
Example: cd C:\Users\YourName\Documents\website

तपाईंको फोल्डरमा जान cd कमाण्ड प्रयोग गर्नुहोस्:
उदाहरण: cd C:\Users\YourName\Documents\website

Step 3: Start Python Server
तेस्रो चरण: पाइथन सर्भर सुरु गर्नुहोस्

Run this command to start a local server:

स्थानीय सर्भर सुरु गर्न यो कमाण्ड चलाउनुहोस्:

python -m http.server 8000

Step 4: Find Your IP Address
चौथो चरण: तपाईंको IP ठेगाना फेला पार्नुहोस्

1. Open a new Command Prompt window
2. Type "ipconfig" and press Enter
3. Look for "IPv4 Address" under your active network adapter
(Usually starts with 192.168. or 10.0.)

१. नयाँ कमाण्ड प्रम्प्ट विन्डो खोल्नुहोस्
२. "ipconfig" टाइप गर्नुहोस् र Enter थिच्नुहोस्
३. तपाईंको सक्रिय नेटवर्क एडाप्टर अन्तर्गत "IPv4 Address" खोज्नुहोस्
(साधारणतया 192.168. वा 10.0. बाट सुरु हुन्छ)

Step 5: Access Your Page
पाँचौं चरण: तपाईंको पेज पहुँच गर्नुहोस्

1. Open a web browser
2. Type your IP address followed by :8000
Example: http://192.168.1.5:8000
3. Click on your HTML file name in the directory listing

१. वेब ब्राउजर खोल्नुहोस्
२. तपाईंको IP ठेगाना र त्यसपछि :8000 टाइप गर्नुहोस्
उदाहरण: http://192.168.1.5:8000
३. डाइरेक्टरी सूचीमा तपाईंको HTML फाइल नाममा क्लिक गर्नुहोस्

Note: Other devices on your local network can now access your page using your IP address. To stop the server, press Ctrl+C in the Command Prompt.

नोट: तपाईंको स्थानीय नेटवर्कमा भएका अन्य उपकरणहरूले अब तपाईंको IP ठेगाना प्रयोग गरेर तपाईंको पेज पहुँच गर्न सक्छन्। सर्भर रोक्न, कमाण्ड प्रम्प्टमा Ctrl+C थिच्नुहोस्।
