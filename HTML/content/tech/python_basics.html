Python Programming Basics / पाइथन प्रोग्रामिङ आधारभूत

Variables / चरहरू

In Python, you can declare variables without specifying their type:

पाइथनमा, तपाईं चरहरूको प्रकार निर्दिष्ट नगरी घोषणा गर्न सक्नुहुन्छ:

name = "Ram"
age = 25
height = 1.75
is_student = True

Numbers and Calculations / संख्याहरू र गणनाहरू

Python supports various mathematical operations:

पाइथनले विभिन्न गणितीय कार्यहरूलाई समर्थन गर्दछ:

# Basic operations / आधारभूत कार्यहरू
addition = 5 + 3        # 8
subtraction = 10 - 4    # 6
multiplication = 4 * 3  # 12
division = 15 / 3       # 5.0
floor_division = 17 // 3  # 5
modulus = 17 % 3        # 2
power = 2 ** 3         # 8

# Float operations / दशमलव संख्या कार्यहरू
pi = 3.14159
rounded = round(pi, 2)  # 3.14

Strings / स्ट्रिङहरू

Strings are sequences of characters:

स्ट्रिङहरू अक्षरहरूको अनुक्रम हुन्:

# String declaration / स्ट्रिङ घोषणा
greeting = "नमस्ते"
message = 'Hello, World!'

# String concatenation / स्ट्रिङ जोड्ने
full_greeting = greeting + " " + message  # नमस्ते Hello, World!

F-strings and Variable Interpolation / एफ-स्ट्रिङ र चर अन्तर्वेशन

F-strings make it easy to embed expressions inside string literals:

एफ-स्ट्रिङले स्ट्रिङ भित्र चरहरू राख्न सजिलो बनाउँछ:

name = "Sita"
age = 30

# Using f-strings / एफ-स्ट्रिङको प्रयोग
message = f"My name is {name} and I am {age} years old"
nepali_message = f"मेरो नाम {name} हो र म {age} वर्षको छु"

Slices / स्लाइसहरू

Slicing allows you to extract parts of sequences:

स्लाइसिङले तपाईंलाई अनुक्रमको भागहरू निकाल्न अनुमति दिन्छ:

text = "Python Programming"
first_word = text[0:6]    # "Python"
last_word = text[7:]      # "Programming"
reversed = text[::-1]     # "gnimmargorP nohtyP"

Lists / सूचीहरू

Lists are ordered collections that can hold different types of data:

सूचीहरू क्रमबद्ध संग्रह हुन् जसले विभिन्न प्रकारका डाटा राख्न सक्छन्:

# Creating lists / सूची बनाउने
fruits = ["Apple", "Banana", "Orange"]
mixed = [1, "Hello", 3.14, True]

# List operations / सूची कार्यहरू
fruits.append("Mango")     # Add item / वस्तु थप्ने
fruits.remove("Banana")    # Remove item / वस्तु हटाउने
first_fruit = fruits[0]    # Access item / वस्तु पहुँच
length = len(fruits)       # List length / सूचीको लम्बाई

Dictionaries / शब्दकोशहरू

Dictionaries store key-value pairs:

शब्दकोशहरूले key-value जोडीहरू भण्डारण गर्छन्:

# Creating dictionary / शब्दकोश बनाउने
person = {
    "name": "Hari",
    "age": 28,
    "city": "Kathmandu",
    "is_student": False
}

# Dictionary operations / शब्दकोश कार्यहरू
name = person["name"]          # Access value / मान पहुँच
person["phone"] = "9841234567" # Add new pair / नयाँ जोडी थप्ने
del person["age"]             # Remove pair / जोडी हटाउने
keys = person.keys()          # Get all keys / सबै keys प्राप्त
