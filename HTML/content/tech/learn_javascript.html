Introduction to JavaScript
जाभास्क्रिप्ट परिचय

What is JavaScript?
जाभास्क्रिप्ट के हो?

JavaScript is a programming language that makes web pages interactive. It can respond to user actions, modify HTML content, and change styles dynamically.

जाभास्क्रिप्ट एक प्रोग्रामिङ भाषा हो जसले वेब पेजहरूलाई इन्टरएक्टिभ बनाउँछ। यसले प्रयोगकर्ता कार्यहरूको प्रतिक्रिया गर्न, HTML सामग्री परिमार्जन गर्न र शैलीहरू गतिशील रूपमा परिवर्तन गर्न सक्छ।

Basic HTML Page
आधारभूत HTML पेज

Let's start with a simple HTML page:

एउटा सरल HTML पेजबाट सुरु गरौं:

<!DOCTYPE html> <html> <head> <title>JavaScript Demo</title> </head> <body> <h1>Welcome to JavaScript Demo</h1> <button id="alertButton">Click for Alert</button> <button id="colorButton">Change Background</button> </body> </html>

Adding JavaScript
जाभास्क्रिप्ट थप्दै

Method 1: Internal JavaScript
विधि १: आन्तरिक जाभास्क्रिप्ट

Add JavaScript inside the HTML file using the <script> tag:

<script> ट्याग प्रयोग गरेर HTML फाइल भित्र जाभास्क्रिप्ट थप्नुहोस्:

<!DOCTYPE html> <html> <head> <title>JavaScript Demo</title> </head> <body> <h1>Welcome to JavaScript Demo</h1> <button id="alertButton">Click for Alert</button> <button id="colorButton">Change Background</button> <script> // Add click event to alert button document.getElementById('alertButton').onclick = function() { alert('Hello! You clicked the button!'); }; // Add click event to color button document.getElementById('colorButton').onclick = function() { document.body.style.backgroundColor = '#' + Math.floor(Math.random()*16777215).toString(16); }; </script> </body> </html>

Method 2: External JavaScript
विधि २: बाह्य जाभास्क्रिप्ट

Create a separate file named script.js:

script.js नामको छुट्टै फाइल सिर्जना गर्नुहोस्:

// script.js document.getElementById('alertButton').onclick = function() { alert('Hello! You clicked the button!'); }; document.getElementById('colorButton').onclick = function() { document.body.style.backgroundColor = '#' + Math.floor(Math.random()*16777215).toString(16); };

Then link it in your HTML file:

त्यसपछि यसलाई तपाईंको HTML फाइलमा लिंक गर्नुहोस्:

<script src="script.js"></script>

Code Explanation
कोड व्याख्या

• document.getElementById(): Finds an HTML element by its ID
• onclick: Triggers when element is clicked
• alert(): Shows a popup message
• style.backgroundColor: Changes background color
• Math.random(): Generates random number for color

• document.getElementById(): ID द्वारा HTML तत्व फेला पार्छ
• onclick: तत्व क्लिक गर्दा ट्रिगर हुन्छ
• alert(): पपअप सन्देश देखाउँछ
• style.backgroundColor: पृष्ठभूमि रङ परिवर्तन गर्छ
• Math.random(): रङको लागि र्यान्डम नम्बर जनरेट गर्छ
