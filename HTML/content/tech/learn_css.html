CSS Guide / CSS निर्देशिका

Introduction / परिचय

CSS (Cascading Style Sheets) is a styling language used to describe the presentation of HTML documents.
CSS (क्यास्केडिङ स्टाइल सिट) HTML दस्तावेजहरूको प्रस्तुति वर्णन गर्न प्रयोग गरिने स्टाइलिङ भाषा हो।

Adding CSS to HTML / HTML मा CSS थप्ने

1. Inline CSS / इनलाइन CSS

<p style="color: black; font-size: 16px;">
This paragraph has inline styles.
यो अनुच्छेदमा इनलाइन स्टाइलहरू छन्।
</p>

2. Internal CSS / आन्तरिक CSS

<!DOCTYPE html>
<html>
<head>
<style>
p {
color: black;
font-size: 16px;
}
</style>
</head>
<body>
<p>This paragraph uses internal styles.</p>
<p> यो अनुच्छेदमा इनलाइन स्टाइलहरू </p>
</body>
</html>

3. External CSS / बाह्य CSS

File: styles.css

p {
color: black;
font-size: 16px;
}

File: index.html

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="styles.css">
</head>
<body>
<p>This paragraph uses external styles.</p>
<p>  यो अनुच्छेदले बाह्य स्टाइलहरू प्रयोग गर्छ।   </p>
</body>
</html>

Selectors / सेलेक्टरहरू

/* Element Selector / एलिमेन्ट सेलेक्टर */
p { color: red; }
/* Class Selector / क्लास सेलेक्टर */
.highlight { background-color: yellow; }
/* ID Selector / आईडी सेलेक्टर */
#header { font-size: 24px; }

Box Model / बक्स मोडेल

.box {
margin: 10px;     /* Outer spacing / बाहिरी खाली ठाउँ */
border: 1px solid black;  /* Border / किनारा */
padding: 15px;    /* Inner spacing / भित्री खाली ठाउँ */
width: 300px;     /* Content width / सामग्रीको चौडाई */
height: 200px;    /* Content height / सामग्रीको उचाई */
}

Flexbox / फ्लेक्सबक्स

.container {
display: flex;
justify-content: center;    /* Horizontal alignment / क्षैतिज संरेखण */
align-items: center;        /* Vertical alignment / ठाडो संरेखण */
}

Grid / ग्रिड

.grid-container {
display: grid;
grid-template-columns: 1fr 1fr 1fr;  /* Three equal columns / तीन बराबर स्तम्भहरू */
gap: 10px;                           /* Space between items / वस्तुहरू बीचको खाली ठाउँ */
}
