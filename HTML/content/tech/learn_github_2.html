How to Create a Pull Request / पुल रिक्वेस्ट कसरी सिर्जना गर्ने

Instructions in English

Step 1: Fork the Repository

Go to https://github.com/[organization]/nepal-web and click the "Fork" button in the top-right corner.

Step 2: Clone Your Fork

Clone the repository to your local machine:

git clone https://github.com/[your-username]/nepal-web.git

Step 3: Create a New Branch

Create and switch to a new branch for your changes:

git checkout -b your-feature-name

Step 4: Make Your Changes

Make your desired changes to the content in the HTML directory. Test your changes locally before committing.

Step 5: Commit Your Changes

git add .
git commit -m "Description of your changes"

Step 6: Push to GitHub

git push origin your-feature-name

Step 7: Create Pull Request

Go to the original repository on GitHub and click "New Pull Request". Select your branch and provide a clear description of your changes.

नेपालीमा निर्देशनहरू

चरण १: रिपोजिटरी फोर्क गर्नुहोस्

https://github.com/[organization]/nepal-web मा जानुहोस् र माथि-दायाँ कुनामा रहेको "Fork" बटनमा क्लिक गर्नुहोस्।

चरण २: आफ्नो फोर्क क्लोन गर्नुहोस्

रिपोजिटरीलाई आफ्नो कम्प्युटरमा क्लोन गर्नुहोस्:

git clone https://github.com/[your-username]/nepal-web.git

चरण ३: नयाँ ब्रान्च सिर्जना गर्नुहोस्

आफ्नो परिवर्तनहरूको लागि नयाँ ब्रान्च सिर्जना गर्नुहोस् र त्यसमा स्विच गर्नुहोस्:

git checkout -b your-feature-name

चरण ४: आफ्नो परिवर्तनहरू गर्नुहोस्

HTML डाइरेक्टरीमा आफ्नो इच्छित परिवर्तनहरू गर्नुहोस्। कमिट गर्नु अघि आफ्नो परिवर्तनहरूको स्थानीय परीक्षण गर्नुहोस्।

चरण ५: आफ्नो परिवर्तनहरू कमिट गर्नुहोस्

git add .
git commit -m "तपाईंको परिवर्तनहरूको विवरण"

चरण ६: GitHub मा पुश गर्नुहोस्

git push origin your-feature-name

चरण ७: पुल रिक्वेस्ट सिर्जना गर्नुहोस्

GitHub मा मूल रिपोजिटरीमा जानुहोस् र "New Pull Request" मा क्लिक गर्नुहोस्। आफ्नो ब्रान्च छान्नुहोस् र आफ्नो परिवर्तनहरूको स्पष्ट विवरण प्रदान गर्नुहोस्।
