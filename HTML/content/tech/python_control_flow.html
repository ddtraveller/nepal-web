Python Control Flow and Functions / पाइथन नियन्त्रण प्रवाह र फंक्शनहरू

If Statements / यदि कथन

Conditional statements allow you to execute code based on certain conditions:

सर्तहरूको आधारमा कोड कार्यान्वयन गर्न सर्त कथनहरू प्रयोग गरिन्छ:

age = 18
if age < 13:
    print("Child")
elif age < 20:
    print("Teenager")
else:
    print("Adult")

# Multiple conditions / बहु सर्तहरू
is_student = True
has_id = True
if is_student and has_id:
    print("Can enter library")
elif is_student or has_id:
    print("Need additional verification")

For Statements / फर लूप कथन

For loops are used to iterate over sequences:

फर लूपहरू अनुक्रमहरूमा पुनरावृत्ति गर्न प्रयोग गरिन्छ:

# Basic for loop / आधारभूत फर लूप
fruits = ["Apple", "Banana", "Orange"]
for fruit in fruits:
    print(fruit)

# Nested for loop / नेस्टेड फर लूप
for i in range(3):
    for j in range(2):
        print(f"i={i}, j={j}")

# Dictionary iteration / शब्दकोश पुनरावृत्ति
person = {"name": "Hari", "age": 25}
for key, value in person.items():
    print(f"{key}: {value}")

The range() Function / range() फंक्शन

range() generates a sequence of numbers:

range() ले संख्याहरूको अनुक्रम उत्पन्न गर्दछ:

# range(stop) / रोक्ने संख्या
for i in range(5):  # 0, 1, 2, 3, 4
    print(i)

# range(start, stop) / सुरु र रोक्ने संख्या
for i in range(2, 5):  # 2, 3, 4
    print(i)

# range(start, stop, step) / सुरु, रोक्ने र कदम संख्या
for i in range(0, 10, 2):  # 0, 2, 4, 6, 8
    print(i)

Break and Continue Statements / ब्रेक र कन्टिन्यु कथन

Break exits the loop, while continue skips to the next iteration:

ब्रेकले लूप बाहिर निस्कन्छ, जबकि कन्टिन्युले अर्को पुनरावृत्तिमा जान्छ:

# Break example / ब्रेक उदाहरण
for i in range(10):
    if i == 5:
        break
    print(i)  # Prints 0 to 4

# Continue example / कन्टिन्यु उदाहरण
for i in range(5):
    if i == 2:
        continue
    print(i)  # Prints 0, 1, 3, 4

Else Clauses on Loops / लूपमा एल्स क्लज

The else clause executes when the loop completes normally (without break):

एल्स क्लज तब कार्यान्वयन हुन्छ जब लूप सामान्य रूपमा पूरा हुन्छ (ब्रेक बिना):

# Loop with else / एल्स सहितको लूप
for i in range(5):
    print(i)
else:
    print("Loop completed!")

# Break prevents else / ब्रेकले एल्स रोक्छ
for i in range(5):
    if i == 3:
        break
    print(i)
else:
    print("This won't print")

The pass Statement / पास कथन

pass is a null operation used when a statement is required syntactically:

पास एक खाली कार्य हो जुन कथन व्याकरणिक रूपमा आवश्यक हुँदा प्रयोग गरिन्छ:

# Empty function / खाली फंक्शन
def not_implemented_yet():
    pass

# Empty class / खाली क्लास
class EmptyClass:
    pass

# Placeholder in if statement / यदि कथनमा प्लेसहोल्डर
if True:
    pass  # Will implement later

Defining Functions / फंक्शन परिभाषा

Functions are defined using the def keyword:

फंक्शनहरू def कीवर्ड प्रयोग गरेर परिभाषित गरिन्छ:

# Basic function / आधारभूत फंक्शन
def greet(name):
    return f"Hello, {name}!"

# Function with default parameter / पूर्वनिर्धारित प्यारामिटर सहितको फंक्शन
def calculate_power(base, exponent=2):
    return base ** exponent

# Function with multiple returns / बहु रिटर्न सहितको फंक्शन
def check_number(n):
    if n > 0:
        return "Positive"
    elif n < 0:
        return "Negative"
    return "Zero"

Main Block / मुख्य ब्लक

The if __name__ == '__main__' idiom is used to control code execution when the module is run directly:

मोड्युल सिधै चलाइँदा कोड कार्यान्वयन नियन्त्रण गर्न if __name__ == '__main__' प्रयोग गरिन्छ:

def main():
    print("Main function executing")
    # Main program logic here
    
if __name__ == '__main__':
    main()  # Only runs if script is run directly

# Example with multiple functions / बहु फंक्शन सहितको उदाहरण
def setup():
    print("Setting up resources")

def cleanup():
    print("Cleaning up resources")

if __name__ == '__main__':
    setup()
    main()
    cleanup()

Note: When a Python file is run directly, __name__ is set to '__main__'. When it's imported as a module, __name__ is set to the module's name.

नोट: जब पाइथन फाइल सिधै चलाइन्छ, __name__ '__main__' मा सेट हुन्छ। जब यो मोड्युलको रूपमा आयात गरिन्छ, __name__ मोड्युलको नाममा सेट हुन्छ।
