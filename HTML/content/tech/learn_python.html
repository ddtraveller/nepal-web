What is Python?
पाइथन के हो?

Python is a popular, easy-to-learn programming language known for its simple syntax and wide range of applications.

पाइथन एक लोकप्रिय, सिक्न सजिलो प्रोग्रामिङ भाषा हो जुन यसको सरल सिन्ट्याक्स र विभिन्न प्रयोगहरूको लागि प्रसिद्ध छ।

Installing Python
पाइथन इन्स्टल गर्दै

Step 1: Download Python
पहिलो चरण: पाइथन डाउनलोड गर्नुहोस्

1. Visit python.org/downloads
2. Click "Download Python" (latest version)
3. Make sure to check "Add Python to PATH" during installation

१. python.org/downloads मा जानुहोस्
२. "Download Python" मा क्लिक गर्नुहोस् (नवीनतम संस्करण)
३. इन्स्टलेसन समयमा "Add Python to PATH" मा चिन्ह लगाउन नबिर्सनुहोस्

Important: The "Add Python to PATH" option is crucial for using Python from the command line.

महत्त्वपूर्ण: कमाण्ड लाइनबाट पाइथन प्रयोग गर्न "Add Python to PATH" विकल्प महत्त्वपूर्ण छ।

Python IDEs (Development Environments)
पाइथन IDEs (डेभलपमेन्ट वातावरणहरू)

1. IDLE (Comes with Python)
१. IDLE (पाइथनसँगै आउँछ)

• Basic IDE included with Python installation
• Good for beginners
• Start from Windows menu → Python → IDLE

• पाइथन इन्स्टलेसनसँग आउने आधारभूत IDE
• सुरुवातकर्ताहरूको लागि राम्रो
• Windows मेनु → Python → IDLE बाट सुरु गर्नुहोस्

2. Visual Studio Code (Free)
२. भिजुअल स्टुडियो कोड (नि:शुल्क)

• Download from code.visualstudio.com
• Install Python extension from VS Code marketplace
• Popular among professionals

• code.visualstudio.com बाट डाउनलोड गर्नुहोस्
• VS Code मार्केटप्लेसबाट पाइथन एक्सटेन्सन इन्स्टल गर्नुहोस्
• पेशेवरहरू माझ लोकप्रिय

3. PyCharm Community Edition (Free)
३. पाइचार्म कम्युनिटी एडिसन (नि:शुल्क)

• Download from jetbrains.com/pycharm
• Full-featured Python IDE
• Includes debugging tools

• jetbrains.com/pycharm बाट डाउनलोड गर्नुहोस्
• पूर्ण सुविधायुक्त पाइथन IDE
• डिबगिङ टूलहरू समावेश छन्

Using Python in Windows
Windows मा पाइथन प्रयोग गर्दै

Method 1: Interactive Mode (Command Line)
विधि १: इन्टर्याक्टिभ मोड (कमाण्ड लाइन)

1. Open Command Prompt (Windows + R, type "cmd")
2. Type "python" and press Enter
3. You'll see the Python prompt ">>>"

१. कमाण्ड प्रम्प्ट खोल्नुहोस् (Windows + R, "cmd" टाइप गर्नुहोस्)
२. "python" टाइप गर्नुहोस् र Enter थिच्नुहोस्
३. तपाईंले पाइथन प्रम्प्ट ">>>" देख्नुहुनेछ

C:\Users\YourName> python >>>

Method 2: Running Python Scripts
विधि २: पाइथन स्क्रिप्टहरू चलाउँदै

1. Create a .py file (e.g., program.py)
2. Open Command Prompt in the file's directory
3. Run: python program.py

१. .py फाइल सिर्जना गर्नुहोस् (जस्तै, program.py)
२. फाइलको डाइरेक्टरीमा कमाण्ड प्रम्प्ट खोल्नुहोस्
३. चलाउनुहोस्: python program.py

Your First Python Program
तपाईंको पहिलो पाइथन प्रोग्राम

print("Hello, World!")

Code Explanation
कोड व्याख्या

• print() is a built-in Python function
• The text in quotes ("Hello, World!") is called a string
• Parentheses () are used to call the function
• The program outputs: Hello, World!

• print() एउटा बिल्ट-इन पाइथन फंक्सन हो
• उद्धरण चिह्नभित्रको पाठ ("Hello, World!") लाई स्ट्रिङ भनिन्छ
• फंक्सन कल गर्न कोष्ठकहरू () प्रयोग गरिन्छ
• प्रोग्रामले यो आउटपुट दिन्छ: Hello, World!

Try running this program both in interactive mode and as a script!

यो प्रोग्रामलाई इन्टर्याक्टिभ मोड र स्क्रिप्टको रूपमा दुवै तरिकाले चलाएर हेर्नुहोस्!
