जुपाइटर ल्याब्स परिचय

Introduction to Jupyter Labs

जुपाइटर ल्याब्स नोटबुक, कोड र डाटासँग काम गर्न एक शक्तिशाली, अर्को पुस्ताको वेब-आधारित इन्टरएक्टिभ विकास वातावरण हो। यो क्लासिक जुपाइटर नोटबुकको विकास हो, जसले डाटा वैज्ञानिक, अनुसन्धानकर्ता र विकासकर्ताहरूका लागि अधिक लचिलो र सुविधा सम्पन्न कार्यस्थल प्रदान गर्छ।

जुपाइटर ल्याब्सका मुख्य विशेषताहरू

Key Features of Jupyter Labs
  • एकीकृत विकास वातावरण / Integrated Development Environment: नोटबुक, पाठ सम्पादक, टर्मिनल, डाटा फाइल दर्शक र अन्य धेरै उपकरणहरू समावेश गर्ने एक व्यापक कार्यस्थल प्रदान गर्छ।
  • लचिलो लेआउट / Flexible Layout: ड्र्याग र ड्रप कार्यक्षमताको साथ धेरै कागजात र गतिविधिहरू एकै साथ मिलाउन सक्छ।
  • बहु प्रोग्रामिङ भाषाहरूको समर्थन / Support for Multiple Programming Languages: पाइथन, आर, जुलिया लगायत ४० भन्दा बढी प्रोग्रामिङ भाषाहरूलाई समर्थन गर्छ।
  • समृद्ध पारिस्थितिकी तन्त्र / Rich Ecosystem: विभिन्न वैज्ञानिक कम्प्युटिङ लाइब्रेरी र डाटा भिजुअलाइजेसन उपकरणहरूसँग एकीकृत हुन्छ।

इन्स्टलेसन गाइड

Installation Guide

पूर्व आवश्यकताहरू

Prerequisites

जुपाइटर ल्याब्स इन्स्टल गर्नु अघि, तपाईंसँग पाइथन र पिप (पाइथन प्याकेज म्यानेजर) इन्स्टल गरिएको हुनुपर्छ। यहाँ इन्स्टलेसनका विधिहरू छन्:

विधि १: पिप मार्फत इन्स्टलेसन

Method 1: Installing via Pip (Recommended) 
pip install jupyterlab

विधि २: कोन्डा मार्फत इन्स्टलेसन

Method 2: Installing via Conda (Recommended for Data Science) 
conda install -c conda-forge jupyterlab

विधि ३: अनाकोन्डा वितरण मार्फत इन्स्टलेसन

Method 3: Installing via Anaconda Distribution

यदि तपाईं अनाकोन्डा प्रयोग गर्दै हुनुहुन्छ भने, जुपाइटर ल्याब्स पहिले नै इन्स्टल गरिएको हुन्छ। अनाकोन्डा navigator बाट यसलाई सुरु गर्नुहोस्।

जुपाइटर ल्याब्स सञ्चालन गर्ने

Running Jupyter Labs

इन्स्टलेसन पछि, तपाईं आफ्नो टर्मिनल वा कमाण्ड प्रोम्प्टमा निम्न कमाण्ड प्रयोग गरेर जुपाइटर ल्याब्स सुरु गर्न सक्नुहुन्छ:

jupyter lab

जुपाइटर सेटअप र प्रयोग गाइड

Comprehensive Jupyter Setup and Usage Guide 
# पाइथन इन्स्टलेसन (यदि पहिले नै इन्स्टल गरिएको छैन भने)
# Windows/Mac/Linux का लागि
# python.org बाट डाउनलोड गर्नुहोस् वा प्याकेज म्यानेजर प्रयोग गर्नुहोस्

# pip (पाइथन प्याकेज म्यानेजर) इन्स्टल गर्नुहोस्
python -m ensurepip --upgrade

# भर्चुअल वातावरण सिर्जना गर्नुहोस् (अनुशंसित)
python -m venv jupyter_env

# भर्चुअल वातावरण सक्रिय गर्नुहोस्
# Windows
jupyter_env\Scripts\activate
# Mac/Linux
source jupyter_env/bin/activate

# जुपाइटर मुख्य प्याकेजहरू इन्स्टल गर्नुहोस्
pip install jupyter jupyterlab notebook

# अतिरिक्त वैज्ञानिक कम्प्युटिङ प्याकेजहरू इन्स्टल गर्नुहोस्
pip install numpy pandas matplotlib seaborn scikit-learn

# जुपाइटर ल्याब सुरु गर्नुहोस्
jupyter lab

# वैकल्पिक: क्लासिक नोटबुक इन्टरफेस सुरु गर्नुहोस्
jupyter notebook

# सामान्य जुपाइटर कमाण्डहरू
# इन्स्टल गरिएका एक्सटेन्सनहरू सूची
jupyter labextension list

# जुपाइटर ल्याब एक्सटेन्सन इन्स्टल गर्नुहोस्
jupyter labextension install extension_name

# जुपाइटर ल्याब अपग्रेड गर्नुहोस्
pip install --upgrade jupyterlab

# नयाँ नोटबुक सिर्जना गर्ने
# 1. जुपाइटर ल्याब इन्टरफेसमा '+' क्लिक गर्नुहोस्
# 2. इच्छित केर्नेल (पाइथन ३, आदि) छान्नुहोस्

# आधारभूत नोटबुक सञ्चालनहरू
# Shift + Enter: वर्तमान सेल सञ्चालन गर्ने
# Esc: कमाण्ड मोड
# Enter: सम्पादन मोड
# 'B': तल सेल थप्ने
# 'A': माथि सेल थप्ने
# 'DD': वर्तमान सेल मेटाउने

# उदाहरण: आधारभूत पाइथन नोटबुक सेटअप
# सेल १: लाइब्रेरीहरू आयात गर्ने
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# सेल २: नमुना डाटा सिर्जना गर्ने
data = np.random.randn(100)

# सेल ३: डाटा भिजुअलाइज गर्ने
plt.figure(figsize=(10,6))
plt.hist(data, bins=20)
plt.title('यादृच्छिक डाटा वितरण')
plt.show()

Creating a New Notebook
  1. लन्चरमा "+" आइकनमा क्लिक गर्नुहोस् / Click on the "+" icon in the launcher
  2. इच्छित केर्नेल (प्रोग्रामिङ भाषा) छान्नुहोस् / Select the desired kernel (programming language)
  3. सेलहरूमा आफ्नो कोड लेख्न सुरु गर्नुहोस् / Start writing your code in cells

सेल प्रकारहरू

Cell Types
  • कोड सेलहरू / Code Cells: जहाँ तपाईं सञ्चालन योग्य कोड लेख्न सक्नुहोस्
  • मार्कडाउन सेलहरू / Markdown Cells: डकुमेन्टेसन, व्याख्या र समृद्ध पाठ ढाँचाका लागि
  • कच्चा सेलहरू / Raw Cells: अस्वरूपित पाठका लागि
