Introduction to Linux Command Line and Bash

लिनक्स कमाण्ड लाइन र बास परिचय

What is the Command Line?

The command line is a text interface for interacting with your computer's operating system.

कमाण्ड लाइन तपाईंको कम्प्युटरको अपरेटिङ सिस्टमसँग अन्तरक्रिया गर्न प्रयोग गरिने टेक्स्ट इन्टरफेस हो।

Essential Commands - आवश्यक कमाण्डहरू

1. pwd (Print Working Directory)

pwd

Shows your current directory location

तपाईंको हालको डाइरेक्टरी स्थान देखाउँछ

2. ls (List)

ls ls -l ls -a

Lists files and directories in the current location. Use -l for detailed view and -a to show hidden files.

हालको स्थानमा भएका फाइल र डाइरेक्टरीहरू देखाउँछ। विस्तृत जानकारीको लागि -l र लुकेका फाइलहरू हेर्न -a प्रयोग गर्नुहोस्।

3. cd (Change Directory)

cd directory_name cd .. cd ~

Navigate between directories. Use .. to go up one level, ~ to go to home directory.

डाइरेक्टरीहरू बीच यात्रा गर्न। एक तह माथि जान .. र होम डाइरेक्टरीमा जान ~ प्रयोग गर्नुहोस्।

4. mkdir (Make Directory)

mkdir directory_name

Creates a new directory

नयाँ डाइरेक्टरी सिर्जना गर्छ

5. rm (Remove)

rm filename rm -r directory_name

Deletes files or directories (-r for recursive deletion of directories)

फाइल वा डाइरेक्टरीहरू मेटाउँछ (डाइरेक्टरीहरू मेटाउन -r प्रयोग गर्नुहोस्)

6. cp (Copy)

cp source destination

Copies files or directories from source to destination

फाइल वा डाइरेक्टरीहरूलाई स्रोतबाट गन्तव्यमा प्रतिलिपि गर्छ

7. mv (Move)

mv source destination

Moves or renames files and directories

फाइल र डाइरेक्टरीहरूलाई सार्छ वा पुनःनामाकरण गर्छ

8. cat (Concatenate)

cat filename

Displays file content

फाइलको सामग्री प्रदर्शन गर्छ

9. grep (Global Regular Expression Print)

grep "pattern" filename

Searches for patterns in files

फाइलहरूमा प्याटर्नहरू खोज्छ

10. chmod (Change Mode)

chmod permissions filename

Changes file permissions

फाइल अनुमतिहरू परिवर्तन गर्छ

11. sudo (Super User Do)

sudo command

Executes commands with administrative privileges

प्रशासनिक विशेषाधिकारहरूसँग कमाण्डहरू कार्यान्वयन गर्छ

12. man (Manual)

man command

Shows the manual page for a command

कमाण्डको म्यानुअल पृष्ठ देखाउँछ

Package Managers - प्याकेज म्यानेजरहरू

Ubuntu (apt)

Ubuntu uses the Advanced Package Tool (apt) for managing software packages.

Ubuntu ले सफ्टवेयर प्याकेजहरू व्यवस्थापन गर्न Advanced Package Tool (apt) प्रयोग गर्छ।

sudo apt update sudo apt upgrade sudo apt install package_name sudo apt remove package_name

CentOS (yum)

CentOS uses the Yellowdog Updater Modified (YUM) for package management.

CentOS ले प्याकेज व्यवस्थापनको लागि Yellowdog Updater Modified (YUM) प्रयोग गर्छ।

sudo yum update sudo yum upgrade sudo yum install package_name sudo yum remove package_name

Distribution Differences - वितरण भिन्नताहरू

Ubuntu is based on Debian and uses .deb packages, while CentOS is based on Red Hat Enterprise Linux (RHEL) and uses .rpm packages. Ubuntu is known for its user-friendly interface and extensive community support, making it popular for desktop users. CentOS is preferred in server environments for its stability and long-term support.

Ubuntu डेबियनमा आधारित छ र .deb प्याकेजहरू प्रयोग गर्छ, जबकि CentOS रेड ह्याट एन्टरप्राइज लिनक्स (RHEL) मा आधारित छ र .rpm प्याकेजहरू प्रयोग गर्छ। Ubuntu आफ्नो प्रयोगकर्ता-मैत्री इन्टरफेस र व्यापक समुदाय समर्थनको लागि प्रसिद्ध छ, जसले गर्दा यो डेस्कटप प्रयोगकर्ताहरूको लागि लोकप्रिय छ। CentOS आफ्नो स्थिरता र दीर्घकालीन समर्थनको लागि सर्वर वातावरणमा रुचाइन्छ।
