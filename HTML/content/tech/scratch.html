स्क्र्याच प्रोग्रामिङ सिक्दै

What is Scratch?

Scratch is a visual programming language created by MIT that makes it easy and fun to create interactive stories, games, and animations. It's like building with LEGO blocks, but instead of physical blocks, you use programming blocks that snap together!

While Python is text-based programming where you write code, Scratch uses colorful blocks that you can drag and drop. This makes it an excellent first step before learning Python or other programming languages.

स्क्र्याच के हो?

स्क्र्याच MIT द्वारा सिर्जना गरिएको एक दृश्यात्मक प्रोग्रामिङ भाषा हो जसले अन्तरक्रियात्मक कथाहरू, खेलहरू र एनिमेसनहरू सिर्जना गर्न सजिलो र मजेदार बनाउँछ। यो LEGO ब्लकहरूसँग निर्माण गर्नु जस्तै हो, तर भौतिक ब्लकहरूको सट्टा तपाईंले प्रोग्रामिङ ब्लकहरू प्रयोग गर्नुहुन्छ जो एकसाथ जोडिन्छन्!

पाइथन कोड लेख्ने टेक्स्ट-आधारित प्रोग्रामिङ हो भने स्क्र्याचले रङ्गीन ब्लकहरू प्रयोग गर्छ जसलाई तपाईं तान्न र छोड्न सक्नुहुन्छ। यसले पाइथन वा अन्य प्रोग्रामिङ भाषाहरू सिक्नु अघि एक उत्कृष्ट पहिलो कदम बनाउँछ।

What Can You Create with Scratch?

With Scratch, you can create:

  • Interactive stories with characters that talk and move
  • Fun games like catching falling objects or maze adventures
  • Musical projects where sprites dance to the beat
  • Mathematical art with colorful patterns
  • Interactive greeting cards for festivals like Dashain and Tihar

स्क्र्याचले के बनाउन सकिन्छ?

स्क्र्याचले तपाईं निम्न कुराहरू सिर्जना गर्न सक्नुहुन्छ:

  • बोल्ने र चल्ने पात्रहरू भएका अन्तरक्रियात्मक कथाहरू
  • खस्ने वस्तुहरू समात्ने वा भूलभुलैया साहसिक कार्य जस्ता मजेदार खेलहरू
  • स्प्राइटहरू बीटमा नाच्ने संगीतमय परियोजनाहरू
  • रङ्गीन ढाँचाहरू भएको गणितीय कला
  • दशैं र तिहार जस्ता चाडपर्वहरूका लागि अन्तरक्रियात्मक शुभकामना कार्डहरू

Getting Started with Scratch

You can use Scratch in two ways:

  1. Online Editor:
    • Go to scratch.mit.edu
    • Click "Start Creating" to begin a new project
    • No installation needed!
  2. Offline Editor:
    • Download Scratch from scratch.mit.edu/download
    • Install it on your computer
    • Works without internet after installation

स्क्र्याच सुरु गर्ने

तपाईं स्क्र्याच दुई तरिकाले प्रयोग गर्न सक्नुहुन्छ:

  1. अनलाइन एडिटर:
    • scratch.mit.edu मा जानुहोस्
    • नयाँ प्रोजेक्ट सुरु गर्न "Start Creating" मा क्लिक गर्नुहोस्
    • कुनै इन्स्टलेसन आवश्यक छैन!
  2. अफलाइन एडिटर:
    • scratch.mit.edu/download बाट स्क्र्याच डाउनलोड गर्नुहोस्
    • यसलाई आफ्नो कम्प्युटरमा इन्स्टल गर्नुहोस्
    • इन्स्टलेसन पछि इन्टरनेट बिना काम गर्छ

Your First Scratch Program: Dancing Cat!

Let's create a simple program where a cat dances and changes colors when you click it!

1. Start with the cat sprite (automatically provided)

2. Add these blocks:

  • When this sprite clicked
  • Forever loop containing:
    • Move 10 steps
    • If on edge, bounce
    • Change color effect by 25
    • Wait 0.2 seconds

तपाईंको पहिलो स्क्र्याच प्रोग्राम: नाच्ने बिरालो!

आउनुहोस् एउटा सरल प्रोग्राम बनाउँ जहाँ बिरालोले नाच्छ र क्लिक गर्दा रङहरू परिवर्तन गर्छ!

1. बिरालो स्प्राइटबाट सुरु गर्नुहोस् (स्वचालित रूपमा प्रदान गरिएको)

2. यी ब्लकहरू थप्नुहोस्:

  • जब यो स्प्राइट क्लिक गरिन्छ
  • सधैंको लागि लूप समावेश:
    • 10 कदम सर्नुहोस्
    • यदि किनारामा छ भने, उफ्रनुहोस्
    • रङ प्रभाव 25 ले परिवर्तन गर्नुहोस्
    • 0.2 सेकेन्ड पर्खनुहोस्

Try It Out!

When you run this program and click the cat, it will start dancing around the screen, bouncing off the edges, and changing colors! This is just the beginning - you can add sounds, more sprites, and make the cat respond to keyboard controls.

प्रयास गर्नुहोस्!

जब तपाईं यो प्रोग्राम चलाउनुहुन्छ र बिरालोमा क्लिक गर्नुहुन्छ, यो स्क्रिनमा वरिपरि नाच्न थाल्नेछ, किनाराहरूमा उफ्रनेछ, र रङहरू परिवर्तन गर्नेछ! यो केवल सुरुवात हो - तपाईं ध्वनिहरू थप्न सक्नुहुन्छ, थप स्प्राइटहरू थप्न सक्नुहुन्छ, र बिरालोलाई किबोर्ड नियन्त्रणहरूमा प्रतिक्रिया गर्न बनाउन सक्नुहुन्छ।
