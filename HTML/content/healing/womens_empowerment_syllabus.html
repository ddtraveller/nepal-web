Women's Empowerment Course - Syllabus

Course Information

Duration: 3 days (21 hours total)
Language: Bilingual instruction (Nepali and English)
Location: Computer lab with internet access

Support Services

  • Childcare available during course hours
  • Transportation assistance if needed
  • Follow-up support after course completion
  • Mentorship connections available

Required Materials

  • Course workbook (provided)
  • USB drive (provided)
  • Basic calculator (provided)
  • Notebook and pen
  • Mobile phone (if available)

Day 1: Digital Literacy and AI Fundamentals

Time Topic Activity Learning Objectives
9:00am - 9:30am Welcome Session Introductions and overview - Understand course structure
- Meet fellow participants
- Set personal goals
9:30am - 10:30am Computer Basics Hands-on practice - Navigate computer interface
- Use basic functions
- Create and save files
10:45am - 12:00pm Internet Safety Interactive workshop - Browse safely online
- Identify secure websites
- Protect personal information
1:00pm - 2:00pm Email Essentials Practical session - Create email account
- Send and receive emails
- Manage attachments
2:00pm - 3:00pm AI Tools Introduction Demonstration and practice - Understand AI capabilities
- Use AI assistants safely
- Practice basic prompts
3:00pm - 4:00pm Mobile Technology Hands-on workshop - Install essential apps
- Use AI-powered features
- Practice mobile safety

Day 2: Financial Literacy and Management

Time Topic Activity Learning Objectives
9:00am - 10:30am Financial Basics Interactive workshop - Understand income/expenses
- Learn saving principles
- Plan financial goals
10:45am - 12:00pm Budgeting Skills Practical exercises - Create personal budget
- Track expenses
- Use budgeting apps
1:00pm - 2:30pm Banking Services Banking workshop - Open bank accounts
- Understand services
- Learn ATM usage
2:45pm - 4:00pm Digital Payments Practical session - Use mobile banking
- Make digital payments
- Practice e-wallet usage

Day 3: Civic Rights and Support Systems

Time Topic Activity Learning Objectives
9:00am - 10:30am Women's Rights Interactive session - Know legal rights
- Understand protections
- Access legal resources
10:45am - 12:00pm Government Services Information workshop - Learn available services
- Access welfare programs
- Complete applications
1:00pm - 2:30pm Support Networks Resource mapping - Identify NGO services
- Connect with resources
- Build support network
2:45pm - 4:00pm Action Planning Planning workshop - Create personal plans
- Set achievable goals
- Establish next steps

Learning Outcomes

Upon completion, participants will be able to:

  • Use computers and internet safely
  • Utilize AI tools appropriately
  • Manage email and digital communication
  • Create and follow personal budgets
  • Use digital banking services
  • Access government support programs
  • Know and advocate for their rights
  • Connect with support resources

Assessment Methods

  • Practical Skills (50%)
    • Computer tasks completion
    • Digital tools usage
    • Financial exercises
  • Participation (30%)
    • Workshop engagement
    • Group discussions
    • Practice activities
  • Action Plan (20%)
    • Personal goal setting
    • Resource identification
    • Implementation strategy

Post-Course Support

  • Monthly follow-up sessions
  • Online support group access
  • Resource updates
  • Technical assistance
  • Mentorship connections

Additional Resources

Participants will receive:

  • Digital resource directory
  • Financial planning templates
  • Government services guide
  • NGO contact information
  • Online learning resources
