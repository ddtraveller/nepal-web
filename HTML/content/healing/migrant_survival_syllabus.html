Migrant Survival and Self-Sufficiency Course - Syllabus

Course Information

Duration: 3 days (21 hours total)
Language: Nepali (with essential phrases in common destination languages)
Location: Various urban and outdoor settings for practical training

Emergency Resources

  • Universal Emergency: 112
  • Nepal Embassy Hotlines (by country)
  • Local Police Numbers
  • Free Medical Centers
  • Community Support Groups

Required Materials

  • Course Handbook:
    • Resource maps
    • Emergency contacts
    • Basic phrase guide
  • Practice Materials:
    • Basic shelter materials
    • Water containers
    • Simple tools
  • Personal Items:
    • Weather-appropriate clothing
    • Small backpack
    • Water bottle

Day 1: Basic Survival Skills

Time Topic Activity Learning Objectives
9:00am - 9:30am Course Introduction Safety briefing - Understand course structure
- Learn safety protocols
- Build trust with group
9:30am - 10:30am Urban Shelter Skills Field practice - Identify safe sleeping locations
- Create temporary shelters
- Use weather protection
10:45am - 12:00pm Water Finding Practical workshop - Locate safe water sources
- Basic water purification
- Public facility access
1:00pm - 2:30pm Food Resources Resource mapping - Find free food sources
- Access food banks
- Safe food storage
2:45pm - 4:00pm Basic Hygiene Practical session - Use public facilities
- Maintain cleanliness
- Access free supplies

Day 2: Communication and Navigation

Time Topic Activity Learning Objectives
9:00am - 10:30am Free Internet Access Location mapping - Find free WiFi spots
- Use public computers
- Access library resources
10:45am - 12:00pm Emergency Communication Practical exercise - Make emergency calls
- Use free calling apps
- Contact embassies
1:00pm - 2:30pm Urban Navigation Field exercise - Use offline maps
- Find free transport
- Navigate safely
2:45pm - 4:00pm Resource Location Community mapping - Find community centers
- Access support services
- Create resource maps

Day 3: Advanced Survival and Community

Time Topic Activity Learning Objectives
9:00am - 10:30am Urban Bushcraft Practical workshop - Create stealth camps
- Find warm spaces
- Manage belongings
10:45am - 12:00pm Emergency Response Scenario training - Handle crisis situations
- Access emergency help
- Practice safety plans
1:00pm - 2:30pm Community Building Network session - Connect with communities
- Share resources
- Build support networks
2:45pm - 4:00pm Integration Skills Practice session - Learn essential phrases
- Understand local systems
- Navigate services

Learning Outcomes

Upon completion, participants will be able to:

  • Basic Survival
    • Find and create safe sleeping spots
    • Access clean water and food
    • Maintain personal hygiene
    • Protect from weather
  • Communication
    • Access free internet and phones
    • Make emergency calls
    • Navigate communication apps
    • Contact support services
  • Resource Access
    • Find community resources
    • Access public facilities
    • Navigate support systems
    • Build community connections

Safety Guidelines

  • Always prioritize personal safety
  • Travel in groups when possible
  • Keep emergency contacts accessible
  • Know local safe zones
  • Maintain awareness of surroundings
  • Trust instincts about danger

Essential Safety Skills

  • Emergency call procedures
  • Quick shelter setup
  • Safe water procurement
  • Basic first aid
  • Crisis response
  • Resource backup plans

Resource Network

  • Nepali community contacts
  • Support organization directory
  • Free service locations
  • Emergency shelter information
  • Medical assistance centers
  • Legal aid resources
