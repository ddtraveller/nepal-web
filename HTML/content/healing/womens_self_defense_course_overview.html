Women's Personal Safety and Self-Defense Course - Nepal

Course Information

Duration: 3 days (18 hours total)
Language: Nepali
Location: Indoor training facility with practice areas

Emergency Resources

  • Nepal Police Women's Cell: 100
  • Women's Helpline: 1145
  • Local Emergency: 112
  • Women's Rights Organizations Contacts
  • Safe House Locations

Required Materials

  • Personal Items:
    • Comfortable, loose-fitting clothes
    • Sports shoes
    • Water bottle
    • Small notebook
  • Course Materials Provided:
    • Safety handbook
    • Emergency contact cards
    • Personal alarm (optional)
    • Practice equipment

Day 1: Awareness and Prevention

Time Topic Activity Learning Objectives
9:00am - 10:00am Course Introduction Interactive discussion - Understand personal safety principles
- Build confidence and awareness
- Learn legal rights
10:15am - 12:00pm Situational Awareness Practical exercises - Identify potential risks
- Read body language
- Practice awareness skills
1:30pm - 3:00pm Verbal Defense Role-playing scenarios - Use assertive communication
- De-escalate confrontations
- Practice boundary setting
3:15pm - 4:30pm Safety Planning Interactive workshop - Create personal safety plans
- Identify safe routes and spaces
- Use safety apps and tools

Day 2: Physical Defense Basics

Time Topic Activity Learning Objectives
9:00am - 10:30am Stance and Movement Physical practice - Learn defensive postures
- Practice safe movement
- Develop stable balance
10:45am - 12:00pm Basic Strikes Technical training - Learn effective strikes
- Practice target areas
- Build striking confidence
1:30pm - 3:00pm Release Techniques Partner practice - Break common holds
- Escape grab attempts
- Use leverage effectively
3:15pm - 4:30pm Ground Defense Floor work - Learn ground positions
- Practice basic escapes
- Maintain safety awareness

Day 3: Integration and Scenarios

Time Topic Activity Learning Objectives
9:00am - 10:30am Defense Review Technique practice - Reinforce core skills
- Build muscle memory
- Increase confidence
10:45am - 12:00pm Multiple Attackers Group scenarios - Understand group dynamics
- Practice escape strategies
- Use environmental awareness
1:30pm - 3:00pm Real-World Scenarios Simulated situations - Apply learned techniques
- Practice decision-making
- Build response confidence
3:15pm - 4:30pm Empowerment Session Group practice and discussion - Share experiences
- Build support network
- Plan ongoing practice

Learning Outcomes

Upon completion, participants will be able to:

  • Personal Safety
    • Assess and avoid potential risks
    • Use situational awareness effectively
    • Implement personal safety plans
    • Access support resources
  • Self-Defense Skills
    • Apply basic defensive techniques
    • Use verbal de-escalation
    • Execute effective escapes
    • Maintain protective stances
  • Confidence Building
    • Trust personal judgment
    • Set clear boundaries
    • React under pressure
    • Access support networks

Safety Guidelines

  • Practice at appropriate intensity
  • Communicate clearly with partners
  • Respect personal boundaries
  • Follow instructor guidance
  • Stop if uncomfortable
  • Support fellow participants

Key Safety Points

  • Prevention is primary defense
  • Trust your instincts
  • Know your surroundings
  • Keep emergency contacts ready
  • Have escape plans prepared
  • Stay connected with support network

Support Network

  • Women's advocacy groups
  • Legal aid resources
  • Community safety organizations
  • Medical support services
  • Mental health resources
  • Course graduate network
